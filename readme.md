# 11ty + Tailwind Example

This is simple example how to combine 11ty + Tailwind CSS + alpinejs

## Installation

Install Postcss globally

```bash
npm i -g postcss
```

Then run

```bash
npm install
```

## Run

```bash
npm run start
```
